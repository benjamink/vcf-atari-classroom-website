Introducing the FujiNet Virtual Machine

The FujiNet Virtual Machine is a pre-built Linux desktop with everything you need to get started actually use FujiNet without needing any hardware other than a reasonably modern computer to run on.

We wanted to provide a way for VCF East attendees to be able to download this rather large (~3.5GB) file without having to worry about a slow Internet connection or using up your valuable cell data allowances. Since you've arrived at this web site you're already on the VCF East local network connection already so downloading the VM file should be relatively painless.

So what do you actually need to run use this VM? It's very simple. The easiest option is to use either Oracle VirtualBox or VMware Workstation Player. Both of these tools are able to import & run the virtual appliance (OVA) version. However, if you prefer to run something less encumbered by commercial ownership you might prefer something like QEMU which can run the QCow2 version. Keep reading below to learn how to download the version that makes the most sense for you.

System Requirements

Each version of the VM requires at least 1 CPU & 4GB of RAM for the guest system itself. Providing the VM with more CPU & especially more RAM will help dramatically improve performance. 2 CPUs & 8GB of RAM is recommended if possible. In addition to the CPU/memory resources you'll need at least 20GB of free hard disk space for the VM's disk itself.
